# NorthStar Aurora Pixel Engine

**Touch-first light show console for professional holiday displays** — Run spectacular synchronized light shows from a handheld device.

---

## Aurora Console

The Aurora Console is a dedicated appliance for creating and controlling RGB pixel light shows, designed for:

- **Professional holiday display operators** — Manage complex multi-controller setups
- **Theme parks and commercial installations** — Reliable, touch-optimized control
- **Hobbyist light show enthusiasts** — AI-powered sequence generation from any song

### Key Features

| Feature | Description |
|---------|-------------|
| **AI Composition** | Generate complete light sequences from music using AI |
| **E1.31/DDP Streaming** | Real-time output to Falcon and compatible controllers |
| **Touch-First Design** | Optimized for ASUS ROG Ally handheld |
| **Internet Archive** | Built-in access to royalty-free music library |
| **Aurora OS** | Purpose-built Linux appliance — boots straight to console |
